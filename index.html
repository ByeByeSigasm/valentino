About the Game

Gorilla Tag is a fun and exciting VR game where players move around by using their arms to run, climb, and tag each other in various game modes. Our version of Gorilla Tag brings enhanced features, new levels, and more customization options!

Game Features

  • Immersive VR experience
  • Multiple game modes: Tag, Infection, and Custom Modes
  • New levels and environments to explore
  • Character customization with skins and accessories
  • Cross-platform multiplayer support
